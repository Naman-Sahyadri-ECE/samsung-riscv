## Components Required:

1.VSD Squadronmini CH32V00x RISC V processor

2.L298N Motor Driver

3.IR Sensor

4.Motors (2 DC motors)

5.Power Supply (12V)

## Pin Connections:

### **IR Sensor**
| Pin | CH32V00x |
|-----|----------|
| VCC | 5V       |
| GND | GND      |
| OUT | PD3      |

### **L298N Motor Driver**
| Pin  | CH32V00x |
|------|----------|
| IN1  | PD1      |
| IN2  | PD2      |
| IN3  | PD4      |
| IN4  | PD7      |
| VCC  | 5V       |
| GND  | GND      |
